Website link - (https://medsumm-kp.lovable.app/)
Drive link - https://drive.google.com/drive/folders/1F6_6j8zt5bKmWb3T2rZA6sw9ctZxMe5n?usp=sharing

## Problem

Patients in India carry physical files of reports to every doctor visit. Doctors waste 50% of consultation time asking for medical history. Allergies get missed. Tests get repeated.

## Solution

MedSumm converts any medical document (photo, PDF, or voice note) into a standardized, color-coded 1-page clinical summary.
